# React / Next.js

## React: 동적 UI

- 컴포넌트 기반 아키텍처:
  - 재사용 가능한 UI 구성 요소
  - 대규모 애플리케이션에서의 일관성 유지
- 단방향 데이터 흐름:
  - 명확한 데이터 관리와 상태 추적
  - 데이터 흐름의 가독성 및 유지보수 용이

## Next.js: SEO 및 성능

- SSR 및 SSG:
  - 빠른 페이지 로드 및 SEO
  - 동적 데이터 fetching 및 렌더링
- 자동 코드 분할 및 최적화:
  - 성능 최적화 및 빠른 로딩 속도
  - 필요한 리소스만 로드하여 효율적 페이지 렌더링
